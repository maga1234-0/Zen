## ❌ Si ça ne marche pas

### Étape 1: Vider complètement le cache
```
1. Ctrl + Shift + Delete
2. Cochez "Images et fichiers en cache"
3. Cliquez "Effacer"
4. Fermez le navigateur
5. Rouvrez et retestez
```

### Étape 2: Vérifier les erreurs
```
1. Appuyez sur F12
2. Onglet "Console"
3. Cherchez les lignes ROUGES
4. Envoyez-moi le message d'erreur
```

### Étape 3: Tester en mode privé
```
1. Ouvrez une fenêtre de navigation privée
2. Allez sur https://zen-lyart.vercel.app
3. Testez les boutons

Si ça marche en mode privé = problème de cache
```
