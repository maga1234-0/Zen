# Icônes Zen Hotel

## Fichiers actuels

- ✅ `zen-icon.svg` - Icône principale (vectorielle, fonctionne pour toutes les tailles)
- ✅ `manifest.json` - Configuration PWA

## Fichiers optionnels (à créer si besoin)

Pour des icônes PNG optimisées, créez:
- `icon-192.png` (192x192px)
- `icon-512.png` (512x512px)
- `apple-touch-icon.png` (180x180px)
- `favicon.ico` (multi-tailles: 16x16, 32x32, 48x48)

## Comment créer les PNG

### Option 1: Générateur automatique (recommandé)
1. Allez sur https://realfavicongenerator.net/
2. Uploadez `zen-icon.svg`
3. Téléchargez et extrayez les fichiers ici

### Option 2: Manuellement
Utilisez un éditeur d'images (Photoshop, GIMP, Figma) pour exporter `zen-icon.svg` aux tailles requises.

## Note
L'icône SVG actuelle fonctionne déjà parfaitement pour tous les navigateurs modernes!
