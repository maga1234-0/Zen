## 🔄 Pour changer l'icône plus tard

Si vous voulez modifier le design:

1. **Modifier le SVG**:
   ```
   Éditez: client/public/zen-icon.svg
   Avec: Inkscape, Figma, ou éditeur de texte
   ```

2. **Ou créer des PNG**:
   ```
   1. Allez sur https://realfavicongenerator.net/
   2. Uploadez votre nouveau design
   3. Téléchargez les fichiers générés
   4. Remplacez dans client/public/
   ```

3. **Pousser sur GitHub**:
   ```bash
   git add client/public
   git commit -m "Update: Nouvelle icône"
   git push origin main
   ```
