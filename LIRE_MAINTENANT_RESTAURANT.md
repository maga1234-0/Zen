## 🧪 TESTER (APRÈS 3 MINUTES)

### 1. Ouvrir l'application
```
https://zen-lyart.vercel.app
```

### 2. Aller au Restaurant
- Cliquez sur **"Restaurant & Bar"** dans le menu

### 3. Créer une commande
- Cliquez sur **"Nouvelle Commande"** (bouton en haut à droite)
- Le formulaire s'ouvre en grand écran ✨

### 4. Remplir le formulaire
- Choisissez le type de service (Chambre, Salle, etc.)
- Sélectionnez une chambre ou table
- Ajoutez des articles du menu
- Cliquez sur **"Créer la Commande"**

## ⚠️ SI ÇA NE MARCHE PAS

### Le formulaire ne s'ouvre pas?
**Solution**: Faites un refresh dur
- Windows: `Ctrl + Shift + R`
- Mac: `Cmd + Shift + R`

### Aucune chambre ou article de menu?
**Solution**: Il faut exécuter le script SQL dans Supabase

1. Ouvrez Supabase Dashboard
2. Allez dans **SQL Editor**
3. Ouvrez le fichier `database/CREER_TABLES_RESTAURANT.sql`
4. Copiez TOUT le contenu
5. Collez dans l'éditeur
6. Cliquez **Run**

Le script crée les tables et ajoute des données de test.
